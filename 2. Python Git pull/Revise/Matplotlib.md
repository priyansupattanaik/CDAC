# Matplotlib Functions

---

# 1. Figure Creation

* `plt.figure()` → Create a new figure
* `plt.subplots()` → Create figure with subplots
* `plt.subplot()` → Add subplot (single)
* `plt.gcf()` → Get current figure
* `plt.gca()` → Get current axes

---

# 2. Basic Plotting

* `plt.plot()` → Line plot
* `plt.scatter()` → Scatter plot
* `plt.bar()` → Bar chart
* `plt.barh()` → Horizontal bar chart
* `plt.hist()` → Histogram
* `plt.pie()` → Pie chart
* `plt.step()` → Step plot
* `plt.stem()` → Stem plot

---

# 3. Advanced Plotting

* `plt.boxplot()` → Box-and-whisker plot
* `plt.violinplot()` → Violin plot
* `plt.errorbar()` → Error bars
* `plt.stackplot()` → Stacked area plot
* `plt.fill_between()` → Area between curves
* `plt.contour()` → Contour plot
* `plt.contourf()` → Filled contour plot
* `plt.imshow()` → Display image data
* `plt.matshow()` → Matrix visualization

---

# 4. Plot Customization

* `plt.title()` → Set plot title
* `plt.xlabel()` / `plt.ylabel()` → Axis labels
* `plt.xlim()` / `plt.ylim()` → Axis limits
* `plt.xticks()` / `plt.yticks()` → Tick control
* `plt.legend()` → Add legend
* `plt.grid()` → Show grid
* `plt.text()` → Add text annotation
* `plt.annotate()` → Annotate points

---

# 5. Styling & Appearance

* `plt.style.use()` → Apply style themes
* `plt.rcParams` → Global settings
* `plt.colorbar()` → Add color scale
* `plt.clim()` → Set color limits
* `plt.tight_layout()` → Adjust layout
* `plt.axis()` → Control axis display

---

# 6. Multiple Plots / Subplots

* `plt.subplots_adjust()` → Adjust spacing
* `fig.add_subplot()` → Add subplot to figure
* `ax.plot()` → Plot on specific axes
* `ax.set_title()` → Set axes title
* `ax.set_xlabel()` / `ax.set_ylabel()` → Axis labels

---

# 7. Saving & Exporting

* `plt.savefig()` → Save figure
* `plt.show()` → Display plot
* `plt.close()` → Close figure

---

# 8. Axis & Scale Control

* `plt.xscale()` / `plt.yscale()` → Log/linear scale
* `plt.twinx()` / `plt.twiny()` → Dual axes
* `plt.invert_xaxis()` / `plt.invert_yaxis()` → Reverse axis

---

# 9. 3D Plotting (`mpl_toolkits.mplot3d`)

* `Axes3D()` → Create 3D axes
* `ax.plot3D()` → 3D line plot
* `ax.scatter3D()` → 3D scatter
* `ax.plot_surface()` → 3D surface plot
* `ax.contour3D()` → 3D contour

---

# 10. Animation (`matplotlib.animation`)

* `FuncAnimation()` → Create animations
* `ArtistAnimation()` → Animation from frames
