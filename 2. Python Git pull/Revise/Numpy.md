# Numpy Function list
---

#  1. ARRAY CREATION

* `np.array()` – Create array
* `np.zeros()` – Array of zeros
* `np.ones()` – Array of ones
* `np.empty()` – Uninitialized array
* `np.arange()` – Range with step
* `np.linspace()` – Evenly spaced values
* `np.eye()` – Identity matrix
* `np.full()` – Fill with constant
* `np.random.rand()` – Random values (uniform)
* `np.random.randn()` – Random values (normal)

---

#  2. ARRAY ATTRIBUTES

* `arr.shape` – Dimensions
* `arr.ndim` – Number of axes
* `arr.size` – Total elements
* `arr.dtype` – Data type

---

#  3. ARRAY MANIPULATION

* `np.reshape()` – Change shape
* `np.ravel()` – Flatten array
* `np.flatten()` – Flatten copy
* `np.transpose()` / `arr.T` – Transpose
* `np.concatenate()` – Join arrays
* `np.vstack()` – Vertical stack
* `np.hstack()` – Horizontal stack
* `np.split()` – Split arrays

* `np.power()`
* `np.sqrt()`

#  5. LINEAR ALGEBRA (`np.linalg`)

* `np.linalg.inv()` – Matrix inverse
* `np.linalg.det()` – Determinant
* `np.linalg.eig()` – Eigenvalues & vectors
* `np.linalg.svd()` – Singular Value Decomposition
* `np.linalg.norm()` – Vector/matrix norm
* `np.linalg.solve()` – Solve linear system
* `np.linalg.matrix_rank()` – Rank
